## 📸 如何添加你的示例图片

你有之前上传的两张图片：
- 原始图片（带背景）
- 处理后的图片（背景已移除）

### 方法 1：通过网站保存（推荐）

1. 访问 https://purebgremover.com
2. 上传原始图片
3. 下载处理后的结果
4. 将图片保存到项目：

```bash
# 将原始图片保存为
public/demo/original.jpg

# 将处理后的图片保存为
public/demo/processed.png
```

### 方法 2：使用你现有的图片

如果你还有原始图片和处理后的图片：

1. 打开项目文件夹 `public/demo/`
2. 将你的图片放入该文件夹
3. 重命名为：
   - `original.jpg` (原始图片，带背景)
   - `processed.png` (处理后的图片，透明背景)

## 🎨 组件功能

### 动画效果
- 初始加载时有淡入和缩放动画
- 背景移除时有闪烁效果（每 5 秒一次）
- 滑块悬停时有放大效果
- 指示箭头有脉冲动画

### 交互
- **桌面端**: 鼠标左右拖动滑块
- **移动端**: 手指左右滑动
- 滑块位置实时更新
- 平滑的过渡效果

### 视觉设计
- 渐变背景（蓝色到紫色）
- 圆角卡片设计
- 阴影效果
- 棋盘格图案表示透明背景
- 标签指示"原始图片"和"背景已移除"

## 🚀 部署步骤

### 1. 添加示例图片（可选）
将你的示例图片放到 `public/demo/` 文件夹：
- `original.jpg` - 原始图片
- `processed.png` - 处理后的图片

### 2. 重新构建
```bash
npm run build
```

### 3. 部署到 Cloudflare Pages
```bash
# 如果使用 Git 自动部署
git add .
git commit -m "Add animated demo component"
git push origin main

# Cloudflare 会自动构建和部署
```

### 4. 验证
访问 https://purebgremover.com，滚动到页面底部查看动画演示效果。

## 📝 图片要求

- **推荐尺寸**: 1280x800 像素 (16:10)
- **格式**: JPG、PNG、WebP
- **文件大小**: < 2MB
- **内容**:
  - 原始图片：包含完整的背景
  - 处理图片：**必须**是透明背景的 PNG

# Demo Images

这个文件夹用于存放着陆页动画演示的示例图片。

## 如何添加自定义图片

1. 将**原始图片**（带背景）保存为 `original.jpg` 或 `original.png`
2. 将**处理后的图片**（背景已移除）保存为 `processed.jpg` 或 `processed.png`

## 图片要求

- **建议尺寸**: 1280x800 像素或类似比例（16:10）
- **格式**: JPG、PNG 或 WebP
- **大小**: 小于 2MB
- **内容**:
  - 原始图片：完整的图片，包含背景
  - 处理图片：主体保留，背景透明

## 当前状态

当前使用默认的 SVG 占位图。添加真实图片后，组件会自动使用它们。

## 获取处理后的图片

你可以通过以下方式获取处理后的图片：

1. 访问 https://purebgremover.com
2. 上传你的原始图片
3. 下载处理后的结果
4. 将两张图片放到这个文件夹中

## 注意

处理后的图片必须是 **透明背景** 的 PNG 格式，才能正确显示效果。
